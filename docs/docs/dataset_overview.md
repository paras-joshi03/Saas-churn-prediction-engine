## Dataset Overview

The project uses the Telco Customer Churn dataset as a reference dataset to design and validate the churn prediction engine.

### Target Variable
- Churn: Indicates whether a customer has discontinued the service.
- Binary classification:
  - 1 → Customer churned
  - 0 → Customer retained

### Feature Categories
- Customer demographics (gender, senior citizen, dependents)
- Service usage information (internet service, streaming services)
- Contract and billing details (contract type, payment method)
- Financial attributes (monthly charges, total charges)
- Tenure (duration of customer relationship)

The dataset represents structured tabular data suitable for supervised machine learning classification tasks.
