# SaaS User Churn Prediction & Retention Engine

This project aims to predict user churn using machine learning techniques and provide actionable retention recommendations. The system is designed as a retrainable churn prediction engine suitable for SaaS platforms.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Streamlit
